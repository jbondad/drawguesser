## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jbondad/drawguesser.git
   ```
3. install modules in front end and back end using
   ```sh
   npm install
   ```
4. Start back end
   ```sh
   node index.js
   ```
5. Start front end using 
   ```sh
   npm start
   ```
6. open http://localhost:3000 to view it in the browser.
